# Image upload in React Vite

    This contains just the front end side of image upload -->
    drag and drop of multiple images and preview of the selected images with cancel feature to unselect an image.

    Packages used for Image upload
    1. Package for Drag and drop  --> npm i react-dropzone
    2. Package for Icons --> npm i react-icons
    3. Package for tailwind css --> npm install tailwindcss @tailwindcss/vite

    For more details regarding dropzone package you can refer the mentioned link ==>( https://www.npmjs.com/package/react-dropzone )
    For icons ==> ( https://www.npmjs.com/package/react-icons )
    For proper installation of Tailwind CSS refer the official website and follow the instructions ==>( https://tailwindcss.com/docs/installation/using-vite )

# To run the react vite 
    -> cd Image_Upload
    -> npm run dev          //then select the local host link
